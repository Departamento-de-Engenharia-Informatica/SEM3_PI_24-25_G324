# Data Dictionary / Glossary Display

This document provides an overview of the tables and their associated fields used in the database.

1. Table: clients
 
| Column        | Type         | Description                                     |
|---------------|--------------|-------------------------------------------------|
| id            | Integer(10)  | Primary Key, Unique identifier for the client.  |
| name          | Varchar(100) | The name of the client.                         |
| address       | Varchar(255) | The physical address of the client.             |
| telephone     | Varchar(20)  | The contact telephone number of the client.     |
| email         | Varchar(100) | The email address of the client.                |
 
2. Table: orders
 
| Column          | Type         | Description                                      |
|-----------------|--------------|--------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the order.    |
| id_client       | Integer(10)  | Foreign Key, References clients(id).             |
| id_operator     | Integer(10)  | Foreign Key, References operators(id).           |
| order_date      | Date         | The date the order was placed.                   |
| deadline        | Date         | The expected deadline for completing the order.  |
 
3. Table: operators
 
| Column          | Type         | Description                                       |
|-----------------|--------------|---------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the operator.  |
| name            | Varchar(100) | The name of the operator.                         |
| specialization  | Varchar(100) | The area of specialization of the operator.       |
| work_shift      | Varchar(50)  | The work shift (e.g., day, night) of the operator.|
 
4. Table: manufacturing_steps
 
| Column          | Type         | Description                                          |
|-----------------|--------------|------------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the step.         |
| name            | Varchar(100) | The name of the manufacturing step.                  |
| duration        | Integer(5)   | The estimated duration of the step (in hours).       |
| status          | Varchar(50)  | The current status (e.g., pending, completed).       |
 
5. Table: products
 
| Column          | Type         | Description                                      |
|-----------------|--------------|--------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the product.  |
| name            | Varchar(100) | The name of the product.                         |
| description     | Varchar(255) | Detailed description of the product.             |
| price           | Decimal(10,2)| The price of the product.                        |
 
6. Table: operations
 
| Column               | Type         | Description                                           |
|----------------------|--------------|-------------------------------------------------------|
| id                   | Integer(10)  | Primary Key, Unique identifier for the operation.     |
| name                 | Varchar(100) | The name of the operation performed.                  |
| id_product           | Integer(10)  | Foreign Key, References products(id).                 |
| id_manufacturing_step| Integer(10)  | Foreign Key, References manufacturing_steps(id).      |
 
7. Table: association_of_operations
 
| Column               | Type         | Description                                          |
|----------------------|--------------|------------------------------------------------------|
| id                   | Integer(10)  | Primary Key, Unique identifier for the association.  |
| id_client            | Integer(10)  | Foreign Key, References clients(id).                 |
| id_product           | Integer(10)  | Foreign Key, References products(id).                |
| id_manufacturing_step| Integer(10)  | Foreign Key, References manufacturing_steps(id).     |
 
8. Table: workstation
 
| Column          | Type         | Description                                      |
|-----------------|--------------|--------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the workstation. |
| location        | Varchar(255) | The physical location of the workstation.        |
| id_operation    | Integer(10)  | Foreign Key, References operations(id).          |
 
9. Table: operation_manufacturing_steps
 
| Column               | Type         | Description                                          |
|----------------------|--------------|------------------------------------------------------|
| id                   | Integer(10)  | Primary Key, Unique identifier for the relationship. |
| id_operation         | Integer(10)  | Foreign Key, References operations(id).              |
| id_manufacturing_step| Integer(10)  | Foreign Key, References manufacturing_steps(id).     |
 
10. Table: operator_has_operations
 
| Column          | Type         | Description                                           |
|-----------------|--------------|-------------------------------------------------------|
| id              | Integer(10)  | Primary Key, Unique identifier for the relationship.  |
| id_operator     | Integer(10)  | Foreign Key, References operators(id).                |
| id_operation    | Integer(10)  | Foreign Key, References operations(id).               |
| id_workstation   | Integer(10)  | Foreign Key, References workstation(id).             |
